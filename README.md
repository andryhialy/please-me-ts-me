# TypeScript Learning Journey

A comprehensive guide to mastering TypeScript through code-along exercises and refresher materials.

## Overview

This repository documents my exploration of TypeScript fundamentals and advanced concepts, with hands-on exercises for each topic.

---

## 1. Getting Started with TypeScript

Master the basics of TypeScript setup and configuration, establishing a solid foundation for all subsequent learning.

---

## 2. Type Fundamentals

Explore primitive types, arrays, tuples, and type handling mechanisms that form the core of TypeScript's type system.

---

## 3. Functions & Interfaces

Understand function typing in TypeScript and use interfaces to define contracts for objects and classes.

---

## 4. Object-Oriented TypeScript

Dive into classes, abstract classes, static members, and leverage TypeScript's OOP capabilities with generics.

---

## 5. Advanced Types

Master sophisticated type techniques including type aliases, unions, intersections, type guards, and utility types.

---

## 6. Modules & Organization

Learn how to structure TypeScript projects using modules and namespaces for maintainability and code reusability.

---

## How to Use This Repository

- Each section contains exercise files and code examples
- Follow along with the exercises to reinforce learning
- Use this as a refresher guide when revisiting TypeScript concepts
